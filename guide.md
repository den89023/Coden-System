Installation and Execution Guide
To run the CODEN system, ensure that Python 3.10 or higher is installed on your environment. The application requires the main execution file (comand.py) and the data storage file (data.json) to be located in the same directory.

Windows Execution
Open the Command Prompt (CMD) or PowerShell.

Navigate to the project folder and execute: python comand.py

Linux Execution
Open the Terminal.

Navigate to the project folder and execute: python3 comand.py
